# Code of Conduct

This project uses a simple standard: be direct, respectful, and constructive.

## Expected Behavior

- Focus criticism on code, documentation, and technical decisions.
- Assume contributors are acting in good faith unless there is clear evidence otherwise.
- Help keep issues and pull requests actionable.

## Unacceptable Behavior

- Harassment, threats, slurs, or personal attacks.
- Publishing private information or credentials.
- Repeated off-topic or hostile comments.

Maintainers may remove comments, close issues, or restrict participation when behavior disrupts the project.
